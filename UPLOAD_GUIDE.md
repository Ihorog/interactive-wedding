# 📸 Керівництво з завантаження медіафайлів

## Як додати фотографії та відео в альбом

### Метод 1: Через AI-чат (Рекомендовано)

1. **Відкрийте AI-помічник**
   - Натисніть на круглу іконку з чатом у правому нижньому куті екрана
   - Вона має фіолетовий/золотавий колір з іконкою чату

2. **Завантажте файли**
   - У відкритому чаті знайдіть іконку скріпки (📎) зліва від поля введення тексту
   - Натисніть на іконку скріпки
   - Оберіть один або кілька файлів (підтримуються фото та відео)
   - Можна обрати багато файлів одночасно (multi-select)

3. **Слідкуйте за прогресом**
   - З'явиться індикатор завантаження з відсотками (0-100%)
   - AI автоматично обробить файли:
     - **Для фото**: стискання для оптимального розміру
     - **Для відео**: створення мініатюри, визначення тривалості

4. **Організуйте контент**
   - Після завантаження натисніть іконку папки (📁) у заголовку чату
   - Відкриється Менеджер медіафайлів
   - Тут ви можете:
     - Призначити фото/відео до розділів (А якщо, Разом, Любити, тощо)
     - Додати назви та описи
     - Додати теги для пошуку
     - Видалити непотрібні файли

### Метод 2: З OneDrive (Якщо зручно)

Якщо ваші фотографії зберігаються на OneDrive:

1. **Отримайте прямі посилання**
   - Відкрийте OneDrive у браузері
   - Оберіть фото/відео
   - Натисніть "Поділитися" → "Копіювати посилання"
   - Переконайтеся, що посилання публічне (доступ для перегляду)

2. **Завантажте через браузер**
   - Відкрийте посилання OneDrive
   - Завантажте файли на свій пристрій
   - Використайте Метод 1 для завантаження в альбом

**Примітка**: Наразі пряма інтеграція з OneDrive не підтримується, тому потрібно спочатку завантажити файли локально.

### Обмеження файлів

- **Максимальний розмір файлу**: 100 МБ
- **Підтримувані формати**:
  - Зображення: JPG, JPEG, PNG, GIF, WebP
  - Відео: MP4, WebM, MOV, AVI
- **Рекомендації**:
  - Для фото: розмір до 5 МБ (автоматично стискається)
  - Для відео: до 50 МБ для оптимальної швидкості

### Організація по розділах

Після завантаження розподіліть медіа по розділах:

- **А якщо** - Підготовка, перші кроки
- **Разом** - Церемонія, розпис, офіційні моменти
- **Любити** - Романтичні моменти
- **Життя** - Неформальні миті
- **По справжньому** - Гості, родичі, вечірка
- **Радіти** - Емоційні піки, танці, сміх
- **Мріяти** - Побажання, майбутнє

### Поради щодо завантаження

✅ **Рекомендується**:
- Завантажуйте по 10-20 файлів за раз для стабільності
- Використовуйте описові назви файлів
- Додавайте теги для легкого пошуку (наприклад: "церемонія", "танець", "гості")
- Першими завантажуйте найкращі фото для швидкого результату

❌ **Уникайте**:
- Дуже великих файлів (>100 МБ)
- Дублікатів фото
- Розмитих або темних знімків низької якості

### Відображення в альбомі

Після організації медіафайли автоматично з'являться:
- У відповідних розділах альбому
- У галереях з lazy loading (поступове завантаження)
- Відео будуть мати мініатюри та кнопку відтворення
- Фото можна відкрити в повноекранному режимі (lightbox)

### Збереження даних

Всі завантажені медіафайли зберігаються:
- **Режим B (Сімейний)**: У хмарному сховищі Spark KV - доступно на всіх пристроях
- **Режим A (Локальний)**: У IndexedDB браузера - тільки на цьому пристрої

Дані зберігаються між сесіями - можете закрити браузер і повернутися пізніше.

## Питання та проблеми

### Файл не завантажується
- Перевірте розмір файлу (максимум 100 МБ)
- Перевірте формат файлу (підтримуються тільки зображення та відео)
- Спробуйте перезавантажити сторінку

### Відео не відтворюється
- Переконайтеся, що формат підтримується браузером (рекомендуємо MP4)
- Перевірте, чи не пошкоджений файл
- Спробуйте завантажити ще раз

### Як видалити файл
1. Відкрийте AI-чат
2. Натисніть іконку папки (Менеджер медіафайлів)
3. Знайдіть потрібний файл
4. Натисніть іконку кошика (🗑️) для видалення

### Файли зникли після перезавантаження
- Якщо використовуєте приватний режим браузера, дані не зберігаються
- Переконайтеся, що не очистили дані браузера
- Використовуйте Режим B (Сімейний) для надійного збереження
